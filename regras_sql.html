Inquérito Relacional

Decifre o banco de dados, encontre o culpado.

// O_CENARIO

Neste desafio baseado no sistema Casos de Código, os times assumem o papel de detetives digitais. Um crime foi cometido e as evidências estão escondidas em um banco de dados relacional complexo.

// A_INVESTIGACAO

Cada equipe receberá uma série de perguntas que só podem ser respondidas através de queries SQL. Cruzar horários, registros de acesso e chaves estrangeiras é a única forma de filtrar os suspeitos e chegar à verdade.

// PONTUACAO

  • Query correta que revela uma pista: 50 créditos.
  • Identificação correta do "culpado" final: 200 créditos.
  • Podem haver bônus por performance.
